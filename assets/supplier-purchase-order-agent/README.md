# Supplier Purchase Order Agent

An AI agent that helps suppliers query open purchase orders and retrieve detailed purchase order information from SAP S/4HANA Cloud

## Overview

Uses A2A Protocol, LangGraph, LiteLLM, and SAP Cloud SDK.

## Structure

- `app/main.py` - A2A server entry
- `app/agent_executor.py` - Request handling
- `app/agent.py` - Agent logic
